Gere uma tabela de limiares de soma para rolagens de dados de 6 faces.

PARÂMETROS (edite só isto):
- Linhas: para cada X de 2 a 9, gere uma linha para Xd6 e outra para Xd6+2 (omita 9d6+2).
- Colunas (alvos de probabilidade): 75%, 50%, 25%.

REGRAS (não mudam):
1. A probabilidade de cada célula é P(soma ≥ N) — a chance de tirar "pelo menos N"
   na soma dos dados (acumulada de cima). Calcule a distribuição EXATA por
   convolução, nunca por aproximação normal.
2. Para cada alvo P e cada linha, escolha o inteiro N que minimiza
   |P(soma ≥ N) − P|. Em empate de distância, prefira o N de MAIOR probabilidade.
3. Em cada célula mostre "N (XX%)", onde XX% é a probabilidade real P(soma ≥ N)
   arredondada ao inteiro.
4. Um modificador fixo (+M) apenas desloca todos os N em +M; ele NÃO altera as
   porcentagens.
5. Saída em tabela markdown. Primeira coluna = nome da rolagem (ex.: "4d6", "4d6+2").
6. Borda do alvo de 75%: se nenhum N cair perto (ex.: o melhor for 72%), siga
   mesmo assim a regra do "mais próximo" do item 2.
